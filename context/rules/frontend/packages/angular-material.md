## Angular Material Theme Tokens (MD3)

* Tokens only: use Angular Material design tokens for colors, typography, spacing, shape, elevation.
* BANNED: HEX/RGB/RGBA, custom font-family, custom font-size.

### [CRITICAL] MD3 Token Mapping

* Material.io uses --md-sys-* tokens.
* Angular Material uses --mat-sys-* tokens for the same roles.
* When using MD3 docs, skills, examples, Figma exports, or generated code, translate roles to the project's Angular Material tokens.
* Never output --md-sys-* in application code.
* Use only tokens defined in the project's theme file.
* Example: --md-sys-color-primary → --mat-sys-primary

### Typography

* Use only typography classes generated by @include mat.system-classes().
* Do not use typography tokens directly.
* Do not define custom font-family or font-size values.

### Material UX

* Follow native Material 3 states, motion, and elevation.
* Prefer Angular Material/CDK behavior over custom implementations.
