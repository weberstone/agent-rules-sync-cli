## Angular Material Theme Tokens
* **Strict Token Use**: Exclusively use design tokens and CSS variables (colors, typography, spacing, elevations).
* **BANNED**: No hardcoded HEX/RGB/RGBA or custom font families/sizes.
* **Material UX Principles**: Strictly follow official Material Design 3 specifications, component states (hover, focus, active), and elevation logic.
* **Component Design**: Leverage native Material Design guidelines and CDK tokens for all custom UI/UX components.
