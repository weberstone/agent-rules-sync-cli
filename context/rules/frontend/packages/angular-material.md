## Angular Material 3 (MD3) Design Tokens
* **Theme & Custom Tokens First:** Use official Angular Material theme variables for colors, spacing, shape, and elevation. Utilizing custom tokens explicitly defined within the project's theme file is permitted.
* **BANNED:** Hardcoded HEX/RGB/RGBA, absolute custom font sizes, and raw font families outside the theme.
* **Token Translation Protocol:** Material.io docs/Figma use `--md-sys-color-primary`. Angular Material strictly uses `--mat-sys-primary`. Globally translate all external snippets: replace `--md-sys-color-[role]` or `--md-sys-[role]` with `--mat-sys-[role]`. Never output `--md-sys-*` prefixes.
* **Typography & UI Engine:** Use typography classes generated via `@include mat.system-classes()`. Do not use tokens directly for fonts. Maximize native Material 3 states, motion, and Angular CDK components over custom implementations.
