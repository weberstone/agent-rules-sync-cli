## Angular Material Theme Tokens
* **Strict Token Use**: Exclusively use design tokens and CSS variables from `src/app/shared/theme` for colors, typography, spacing, and elevations.
* **BANNED: Custom Colors & Fonts**: Never hardcode HEX/RGB/RGBA values or invent custom font sizes/families.
* **Material Principles**: Follow official Material Design UX patterns, component states (hover, focus, pressed), and elevation logic.
